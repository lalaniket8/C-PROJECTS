PARITY - an evenly odd game

A video game in C where you have to click ODD or EVEN depending on the random numbers appearing on the center of the screen.
Player is given 5 Lives.
Each correct answer gives you 100 points.

The highscores are stored in a text file in secondary storage,
each time accessed and updated if user made a highscore.
The same can be reset from the game itself.
